# AKRASC 4.0 Website

A modern, dark-themed, responsive website designed for **AKRASC (All Kerala Robotics & Automation Society Conclave)**. The website is inspired by the official AKRASC platform and presents information about the conclave in a visually engaging and user-friendly format.

## Overview

This project reimagines the AKRASC website with a futuristic design while preserving the original content and purpose. The website focuses on providing visitors with information about the conclave, its objectives, host institution, and venue details through a clean, interactive, and responsive interface.

## Features

* Modern dark-themed user interface
* Fully responsive design for desktop, tablet, and mobile devices
* Smooth scrolling navigation
* Interactive animations and hover effects
* Glassmorphism-inspired design elements
* Dedicated sections for:

  * Home
  * About AKRASC
  * Objectives
  * Event Highlights
  * Host Information
  * Venue Location with Embedded Map
  * Contact Information
* Direct access to the IEEE RAS Kerala Chapter website
* Optimized user experience with modern typography and layouts

## Technologies Used

* HTML5
* CSS3
* Google Fonts
* CSS Animations
* Responsive Design Techniques
